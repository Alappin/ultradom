## Installation

With npm/Yarn.

<pre>
npm i [turbodom](https://npmjs.com/package/turbodom)
</pre>

From a CDN.

```html
<script src="https://unpkg.com/turbodom"></script>
```

## Usage

With ES5/ES6 via a bundler.

```jsx
import { h, patch } from "turbodom"
```

With a <samp>\<script\></samp> tag.

```js
const { h, patch } = turbodom
```
